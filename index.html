featured projects.

Transfer Tracker

Full-stack development

A web application to track football transfer rumours of your favourite team in a simple interface. It uses the Twitter Streaming API to crowd-source the news and updates from all around the globe. Based on your choice of team, this app provides you with tweets made by mostly credible sources and gives you automatic updates in a simple format.

Storally

LOGO DESIGN & Branding

Logo design for a UBC-based startup that aims to help students store their belongings in a safe environment by crowd-sourcing storage space to other students. For this project, I decided to go for a clean, minimal design which uses radiant pastel colours that appeal more to a younder audience and conveys the idea of the startup in a simple way.

Sift

UX Design & development

A web application that gives businesses a birds-eye view into their customers' feedback. It connects to feedback forms, support email, and reviews to create a visual summary of a business's customer relationships using natural language processing. This provides a quick way to see everything important that customers are saying.

Playsmid

Full-stack development

An interactive game that aims to teach synthetic biology to beginners interested in biology. I developed the game engine using jQuery. Also worked on the back-end developed using a Node server which retrieves the game data using a Mongo database.
